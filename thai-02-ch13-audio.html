Overview｜本課總覽

本課主要學什麼？

  1. 用 เอา + N + ไปที่ + place 表達「把某物拿去某地」。
  2. 用 มาเอา + N + ที่ + place 表達「來某地拿某物」。
  3. 用 ช่วยมาเอา...หน่อย 請店員或服務人員幫忙來拿東西。
  4. 用 V + adj + นะคะ / V + ให้ + adj + นะคะ 做提醒或要求。
  5. 用 อย่า + V + adj / อย่า + V + ให้ + adj + นัก 表達「不要太……」。
  6. 學服裝修改、洗衣、裁縫、修車、理髮與做頭髮相關生活詞彙。

曼谷生活會怎麼用？

Day 13 很生活化：在曼谷你會需要拿衣服去洗、去改褲長、請洗衣店來取衣服、去髮廊洗頭剪髮染髮、叫司機慢一點、提醒朋友不要喝太多。這一課不是只背單字，而是要學會「怎麼跟服務人員說需求」。

本課 PDF 對應重點

第 2 頁：S + ไปเอา + N + ที่ + place、มาเอา...กลับ，例句是拿衣服去洗衣店，以及請店員來拿衣服。

第 3 頁：V + adj + นะคะ、V + ให้ + adj + นะคะ，以及 อย่า... 禁止句，例如開車慢一點、不要喝太多。

第 4–5 頁：服裝、選擇、量身、剪、修改、裁縫店、修車、試穿、合身、調整。

第 6–8 頁：理髮店與髮廊：洗頭、剃鬍子、剪頭髮、染頭髮、燙頭髮、拉直、接髮、剪瀏海、包保鮮膜/蒸頭髮。

Review Day 5 學過 ขอ...หน่อยค่ะ / khɔ̌ɔ...nòi khâ / please...。 Day 13 會延伸成更服務業自然的：

Structures｜句型整理

1. S + เอา + N + ไปที่ + Place

Subject + เอา + Noun + ไปที่ + Place
Subject + ao + Noun + bpai thîi + Place
Subject takes Noun to Place.

Textbook-style example

ฉันเอาเสื้อผ้าไปที่ร้านซักรีดค่ะ
chǎn ao sʉ̂a-phâa bpai thîi ráan sák-rîit khâ
I take clothes to the laundry shop.

Life variations

  • ฉันจะเอากางเกงไปแก้ที่ร้านตัดเสื้อค่ะ
    chǎn jà ao gaang-geeng bpai gɛ̂ɛ thîi ráan dtàt-sʉ̂a khâ
    I will take the pants to the tailor shop to alter them.
  • เขาเอารถไปซ่อมที่อู่ค่ะ
    khǎo ao rót bpai sɔ̂ɔm thîi ùu khâ
    He took the car to the garage to repair it.
  • ฉันเอารองเท้าไปซ่อมที่ร้านค่ะ
    chǎn ao rɔɔng-tháao bpai sɔ̂ɔm thîi ráan khâ
    I took the shoes to the shop to repair them.
  • พรุ่งนี้จะเอาชุดไปซักแห้งค่ะ
    phrûng-níi jà ao chút bpai sák hɛ̂ɛng khâ
    Tomorrow I will take the outfit to be dry-cleaned.

2. ช่วย + มาเอา + N + ที่ + Place + หน่อยค่ะ

ช่วย + มาเอา + Noun + ที่ + Place + หน่อยค่ะ
chûai + maa ao + Noun + thîi + Place + nòi khâ
Please come pick up Noun at Place.

Textbook-style example

ช่วยมาเอาเสื้อผ้าที่ร้านหน่อยค่ะ
chûai maa ao sʉ̂a-phâa thîi ráan nòi khâ
Please come pick up the clothes at the shop.

Life variations

  • ช่วยมาเอาเสื้อผ้าที่คอนโดหน่อยค่ะ
    chûai maa ao sʉ̂a-phâa thîi khɔɔn-doo nòi khâ
    Please come pick up the clothes at my condo.
  • ช่วยมาเอาของที่ล็อบบี้หน่อยค่ะ
    chûai maa ao khɔ̌ɔng thîi lɔ́p-bîi nòi khâ
    Please come pick up the things at the lobby.
  • ช่วยมาเอาผ้าที่ห้อง 1208 หน่อยค่ะ
    chûai maa ao phâa thîi hɔ̂ng sìp-sɔ̌ɔng-sǔun-bpɛ̀ɛt nòi khâ
    Please come pick up the laundry at room 1208.
  • ถ้าซักเสร็จแล้ว ช่วยโทรหาฉันหน่อยค่ะ
    thâa sák sèt lɛ́ɛw, chûai thoo hǎa chǎn nòi khâ
    When the laundry is done, please call me.

3. V + adj + นะคะ / V + ให้ + adj + นะคะ

Verb + Adjective + นะคะ / Verb + ให้ + Adjective + นะคะ
Verb + Adjective + ná khá / Verb + hâi + Adjective + ná khá
Please do Verb in an Adj way / Please make it Adj.

Textbook-style examples

  • ขับรถช้า ๆ นะคะ

    khàp-rót hâi cháa ná khá

  • ตัดผมให้สั้นลงนิดหน่อยนะคะ

  • แก้กางเกงให้พอดีนะคะ
    gɛ̂ɛ gaang-geeng hâi phɔɔ-dii ná khá

  • พูดช้า ๆ หน่อยนะคะ
    phûut cháa-cháa nòi ná khá

  • ทำสีให้เข้มขึ้นนิดหน่อยนะคะ
    tham-sǐi hâi khêm khʉ̂n nít-nɔ̀i ná khá
