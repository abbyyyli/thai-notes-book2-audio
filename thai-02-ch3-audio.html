Thai Book 2｜Day 3 學習指南

主題：นะคะ 柔和確認、หรือคะ 表達不同看法、程度副詞、價格/費用/押金/合約、借還、有效期、大量反義詞與同義詞例句

Textbook-based Supplement / Local usage Practical Usage Review Common mistake

Lesson Overview

Textbook-based Day 3 主要在學「怎麼用比較自然的方式表達意見、不同意、程度、價格與反義詞」。教材第 2 頁出現： เมืองไทยร้อนนะคะ / mʉang-thai rɔ́ɔn ná khá / Thailand is hot, you know. 這種 นะคะ / ná khá / soft confirmation particle 不是單純問句，而是像中文「欸，泰國很熱耶 / 對吧」。

如果你不同意，可以用比較柔和的開頭： หรือคะ ฉันคิดว่า... / rʉ̌ʉ khá, chǎn khít wâa... / Really? I think that... 這比直接說 ไม่ใช่ / mâi châi / No, that’s not true 更自然、更禮貌。

Soft statement
ภาษาไทยยากนะคะ
phaa-sǎa Thai yâak ná khá
Thai is difficult, you know.
Soft disagreement
จริงเหรอคะ ฉันว่าไม่ค่อยยากค่ะ
jing rʉ̌ə khá, chǎn wâa mâi khɔ̂i yâak khâ
Really? I think it’s not very difficult.
Degree answer
ไม่แพงเท่าไรค่ะ
mâi phɛɛng thâo-rai khâ
It’s not that expensive.
Contract
สัญญาหมดอายุเดือนหน้าค่ะ
sǎn-yaa mòt aa-yú dʉan nâa khâ
The contract expires next month.
Review： Day 2 已經學過比較級 กว่า / gwàa / more than。Day 3 會用更多形容詞與反義詞來造句，例如： แพงกว่า / phɛɛng gwàa / more expensive than、 ถูกกว่า / thùuk gwàa / cheaper than。
本版加強規則： 同義詞與反義詞不只做表格，兩邊都會給例句，而且例句會放進不同生活情境，例如租屋、交通、點餐、聊天、語言交換、買東西、曼谷生活。

Common Mistakes & Quick Review

Mistake Why It’s Wrong / Risky Better Version
把 นะคะ 當成ไหมคะ
ná khá vs mái khá
soft particle vs question particle
 นะคะ 是柔和確認；ไหมคะ 才是 yes/no question。
เมืองไทยร้อนนะคะ
mʉang-thai rɔ́ɔn ná khá
Thailand is hot, you know.
เมืองไทยร้อนไหมคะ
mʉang-thai rɔ́ɔn mái khá
Is Thailand hot?
不同意時太直接
mâi châi
No / not true
 直接說 ไม่ใช่ 有時很硬。可以先說 จริงเหรอคะ。
จริงเหรอคะ ฉันว่าไม่ค่อยแพงค่ะ
jing rʉ̌ə khá, chǎn wâa mâi khɔ̂i phɛɛng khâ
Really? I think it’s not very expensive.
ถูก 的意思搞混
thùuk
cheap / correct
 ถูก 可以是便宜，也可以是正確，要看上下文。
ราคาถูกค่ะ
raa-khaa thùuk khâ
The price is cheap.
คำตอบถูกค่ะ
kham-dtɔ̀ɔp thùuk khâ
The answer is correct.
แพงเกินไป 用太重
phɛɛng gəən-bpai
too expensive
 เกินไป 比較直接、負面；談價格可用 ไปหน่อย 柔和。
แพงไปหน่อยค่ะ
phɛɛng bpai nɔ̀i khâ
It’s a little too expensive.
ยืม / คืน 方向相反
yʉʉm / khʉʉn
borrow / return
 ยืม 是借來；คืน 是還回去。
ขอยืมหนังสือค่ะ
khɔ̌ɔ yʉʉm nǎng-sʉ̌ʉ khâ
I’d like to borrow the book.
พรุ่งนี้คืนหนังสือค่ะ
phrûng-níi khʉʉn nǎng-sʉ̌ʉ khâ
I’ll return the book tomorrow.
本課最重要記憶點：

1. เมืองไทยร้อนนะคะ / mʉang-thai rɔ́ɔn ná khá / Thailand is hot, you know.

2. จริงเหรอคะ ฉันว่าไม่ค่อยแพงค่ะ / jing rʉ̌ə khá, chǎn wâa mâi khɔ̂i phɛɛng khâ / Really? I think it’s not very expensive.

3. แพงไปหน่อยค่ะ / phɛɛng bpai nɔ̀i khâ / It’s a little too expensive.

4. ต้องวางมัดจำเท่าไรคะ / dtɔ̂ng waang mát-jam thâo-rai khá / How much deposit do I need to put down?

5. สัญญาหมดอายุเดือนหน้าค่ะ / sǎn-yaa mòt aa-yú dʉan nâa khâ / The contract expires next month.
