Thai Book 2｜Day 9 學習指南

主題：ก่อน / หลัง、สาย / ไม่สาย、เหมือนกัน / ด้วยกัน / เท่ากัน / พร้อมกัน / เดียวกัน、交通地點、交通工具與量詞

Textbook-based Supplement / Local usage Practical Usage Review Common mistake

Lesson Overview

Textbook-based Day 9 的主線是「時間順序 + 比較/一起 + 交通」。教材先用 เมื่อก่อน / mʉ̂a-gɔ̀ɔn / before / in the past 和 ตอนนี้ / dtɔɔn-níi / now 對比曼谷以前沒有捷運、現在有捷運，接著進入 ก่อน / gɔ̀ɔn / before 與 หลัง / lǎng / after 的句型。

Before / after
ฉันอาบน้ำก่อนนอนค่ะ
chǎn àap-náam gɔ̀ɔn nɔɔn khâ
I shower before sleeping.
Late / not late
วันนี้ฉันไปเรียนสายค่ะ
wan-níi chǎn bpai rian sǎai khâ
Today I was late for class.
Same / together
เราไปด้วยกันค่ะ
rao bpai dûai-gan khâ
We go together.
Bangkok transport
ฉันขึ้นรถไฟฟ้าที่สถานีโพธิ์นิมิตรค่ะ
chǎn khʉ̂n rót-fai-fáa thîi sà-thǎa-nii phoo-ní-mít khâ
I take the BTS at Pho Nimit station.
Review： Day 8 學過位置詞 ข้างหน้า / ข้างหลัง / khâang-nâa / khâang-lǎng / in front / behind。 Day 9 的 ก่อน / หลัง / gɔ̀ɔn / lǎng / before / after 是時間順序，不是空間前後。
曼谷生活連結： 這課很適合用在 BTS/MRT、巴士、機場、碼頭、火車站情境：問怎麼去、在哪裡上車、會不會遲到、要不要一起去、我們是不是搭同一班車。

Time & Sequence｜時間與先後順序

Textbook-based
เมื่อก่อน
mʉ̂a-gɔ̀ɔn
before / in the past
 Time adverb — 用來對比「以前」和「現在」。比單獨 ก่อน 更像「以前」。
เมื่อก่อน...แต่ตอนนี้... / เมื่อก่อนไม่มี...
mʉ̂a-gɔ̀ɔn...dtɛ̀ɛ dtɔɔn-níi... / mʉ̂a-gɔ̀ɔn mâi-mii...
before...but now... / before there wasn’t...
เมื่อก่อนกรุงเทพฯ ไม่มีรถไฟฟ้าค่ะ
mʉ̂a-gɔ̀ɔn grung-thêep mâi-mii rót-fai-fáa khâ
In the past, Bangkok did not have skytrain/electric train.
Textbook-based
ตอนนี้
dtɔɔn-níi
now
 Time adverb — 表示現在、目前。
ตอนนี้มี... / ตอนนี้อยู่ที่...
dtɔɔn-níi mii... / dtɔɔn-níi yùu thîi...
now there is... / now I am at...
ตอนนี้กรุงเทพฯ มีรถไฟฟ้าหลายสายค่ะ
dtɔɔn-níi grung-thêep mii rót-fai-fáa lǎai sǎai khâ
Now Bangkok has many train lines.
Textbook-based
ก่อน
gɔ̀ɔn
before / first
 Sequence word — 可放在兩個動作中間，表示 V1 在 V2 之前。也可句尾表示「先」。
ก่อนนอน / ก่อนออกจากบ้าน / ไปก่อน
gɔ̀ɔn nɔɔn / gɔ̀ɔn ɔ̀ɔk-jàak bâan / bpai gɔ̀ɔn
before sleeping / before leaving home / go first
ฉันปิดไฟก่อนออกจากบ้านค่ะ
chǎn bpìt fai gɔ̀ɔn ɔ̀ɔk-jàak bâan khâ
I turn off the light before leaving home.
Textbook-based
หลัง
lǎng
after / behind
 Sequence word / position — 本課重點是「after」。Day 8 已學 ข้างหลัง 是 behind。
หลังเลิกเรียน / หลังทำงาน / หลังจาก...
lǎng lə̂ək-rian / lǎng tham-ngaan / lǎng-jàak...
after class / after work / after...
ฉันกินข้าวหลังเลิกเรียนค่ะ
chǎn gin khâao lǎng lə̂ək-rian khâ
I eat after class.
Textbook-based
สาย
sǎai
late
 Time adjective — 表示比預定時間晚。不要和 สาย = line / route 混淆。
ไปสาย / มาสาย / สายแล้ว
bpai sǎai / maa sǎai / sǎai lɛ́ɛw
go late / come late / it’s late already
วันนี้รถติด ฉันเลยมาสายค่ะ
wan-níi rót-dtìt, chǎn ləəi maa sǎai khâ
There was traffic today, so I came late.
Textbook-based
ไม่สาย
mâi sǎai
not late
 Time adjective — 否定放在形容詞前：ไม่ + สาย。
ไปไม่สาย / มาไม่สาย / ยังไม่สาย
bpai mâi sǎai / maa mâi sǎai / yang mâi sǎai
not go late / not come late / still not late
ถ้าออกตอนนี้ เรายังไม่สายค่ะ
thâa ɔ̀ɔk dtɔɔn-níi, rao yang mâi sǎai khâ
If we leave now, we are still not late.

เหมือนกัน
mʉ̌an-gan
same / alike / too
 Comparison — 用來說 N1 跟 N2 一樣，也常用在「我也是」。
สีเหมือนกัน / ชอบเหมือนกัน / เหมือนกันไหม

ฉันกับเพื่อนชอบชาไทยเหมือนกันค่ะ

My friend and I both like Thai tea.
Textbook-based
ด้วยกัน
dûai-gan
together
 Togetherness — 表示一起做某動作。結構常是 N1 กับ N2 + V + ด้วยกัน。
ไปด้วยกัน / กินด้วยกัน / เรียนด้วยกัน
bpai dûai-gan / gin dûai-gan / rian dûai-gan
go together / eat together / study together
เราไปสยามด้วยกันไหมคะ
rao bpai sà-yǎam dûai-gan mái khá
Shall we go to Siam together?
Textbook-based
เท่ากัน
thâo-gan
equal / the same amount
 Quantity comparison — 用於數量、價格、大小、時間長度一樣。
ราคาเท่ากัน / จำนวนเท่ากัน / สูงเท่ากัน
raa-khaa thâo-gan / jam-nuan thâo-gan / sǔung thâo-gan
same price / same quantity / same height
กาแฟสองแก้วนี้ราคาเท่ากันค่ะ
gaa-fɛɛ sɔ̌ɔng gâew níi raa-khaa thâo-gan khâ
These two coffees have the same price.
Textbook-based
พร้อมกัน
phrɔ́ɔm-gan
at the same time / together simultaneously
 Time / action — 重點是「同時發生」，不是單純一起。
ออกพร้อมกัน / ถึงพร้อมกัน / เริ่มพร้อมกัน
ɔ̀ɔk phrɔ́ɔm-gan / thʉ̌ng phrɔ́ɔm-gan / rə̂əm phrɔ́ɔm-gan
leave at the same time / arrive at the same time / start at the same time
เราจะออกจากบ้านพร้อมกันตอนแปดโมงค่ะ
rao jà ɔ̀ɔk-jàak bâan phrɔ́ɔm-gan dtɔɔn bpɛ̀ɛt moong khâ
We will leave home at the same time at 8 o’clock.
Textbook-based
เดียวกัน
diao-gan
the same one / same specific item
 Identity / same specific item — 用於同一個房間、同一瓶水、同一班車。教材提醒：N + Clf. + เดียวกัน。
ห้องเดียวกัน / ขวดเดียวกัน / สายเดียวกัน
hɔ̂ng diao-gan / khùat diao-gan / sǎai diao-gan
same room / same bottle / same line
เราอยู่ห้องเดียวกันค่ะ
rao yùu hɔ̂ng diao-gan khâ
We are in the same room.

Transport & City Facilities｜交通與城市設施

Thai / Romanization / English Category Classifier Grammar Notes Collocations Example Sentence
Textbook-based
ถนน
thà-nǒn
road / street
 City place — 道路名常用 ถนน。在曼谷地址很常見。
ข้ามถนน / ถนนใหญ่ / ถนนสีลม
khâam thà-nǒn / thà-nǒn yài / thà-nǒn sǐi-lom
cross the road / big road / Silom Road
ต้องข้ามถนนตรงทางม้าลายค่ะ

Textbook-based

thaang-máa-laai
zebra crossing / crosswalk
 Road facility — 字面像「斑馬路」。教材頁 7 的路口圖有標示。
ข้ามทางม้าลาย / อยู่ตรงทางม้าลาย
khâam thaang-máa-laai / yùu dtrong thaang-máa-laai

Textbook-based
สะพานลอย
sà-phaan-lɔɔi
pedestrian bridge / overpass
 Road facility สะพาน 曼谷大馬路很常見。比直接穿越馬路安全。
ขึ้นสะพานลอย / ข้ามสะพานลอย / ลงสะพานลอย
khʉ̂n sà-phaan-lɔɔi / khâam sà-phaan-lɔɔi / long sà-phaan-lɔɔi

ถนนนี้รถเยอะ ควรข้ามสะพานลอยค่ะ

There are many cars on this road, so you should cross by the overpass.
Textbook-based
ป้ายรถเมล์
bpâai rót-mee
bus stop
 Transport place ป้าย ป้าย 是牌子、站牌。教材路口圖有站牌。
รอที่ป้ายรถเมล์ / ป้ายรถเมล์อยู่ไหน

wait at the bus stop / where is the bus stop?
ป้ายรถเมล์อยู่หน้าปากซอยค่ะ
bpâai rót-mee yùu nâa bpàak-sɔɔi khâ
The bus stop is at the entrance of the soi.
Textbook-based
ไฟเขียว / ไฟแดง
fai-khǐao / fai-dɛɛng
green light / red light
 Traffic signal — 教材圖中有紅綠燈。ไฟ 是燈；เขียว 綠；แดง 紅。
รอไฟเขียว / ติดไฟแดง / ข้ามตอนไฟเขียว
rɔɔ fai-khǐao / dtìt fai-dɛɛng / khâam dtɔɔn fai-khǐao
wait for green light / stuck at a red light / cross when the light is green
รอไฟเขียวก่อนค่อยข้ามถนนนะคะ
rɔɔ fai-khǐao gɔ̀ɔn khɔ̂i khâam thà-nǒn ná khá
Wait for the green light first, then cross the road.
Textbook-based
คลอง / แม่น้ำ
khlɔɔng / mâe-náam
canal / river
 Waterway — 教材頁 8 有碼頭、คลอง、แม่น้ำ。曼谷交通也會用船。
คลองแสนแสบ / แม่น้ำเจ้าพระยา / ริมแม่น้ำ
khlɔɔng sɛ̌ɛn-sɛ̀ɛp / mâe-náam jâo-phrá-yaa / rim mâe-náam
Saen Saep canal / Chao Phraya River / riverside
วัดอรุณอยู่ริมแม่น้ำเจ้าพระยาค่ะ
wát à-run yùu rim mâe-náam jâo-phrá-yaa khâ
Wat Arun is by the Chao Phraya River.
Textbook-based
ท่าเรือ
thâa-rʉa
pier / boat pier
 Transport place ท่า 搭船地點。曼谷河邊、คลอง 船都會用到。
ขึ้นเรือที่ท่าเรือ / ท่าเรืออยู่ไหน
khʉ̂n rʉa thîi thâa-rʉa / thâa-rʉa yùu nǎi

ท่าเรืออยู่ข้างหลังตลาดค่ะ
thâa-rʉa yùu khâang-lǎng dtà-làat khâ

ก่อน / หลัง｜before / after 的三種用法

Pattern Meaning Textbook-style Example Bangkok Life Example
S + V1 + ก่อน + V2
S + V1 + gɔ̀ɔn + V2
do V1 before V2
 先做 V1，再做 V2。
ฉันอาบน้ำก่อนนอนค่ะ
chǎn àap-náam gɔ̀ɔn nɔɔn khâ

ฉันซื้อกาแฟก่อนขึ้นรถไฟฟ้าค่ะ
chǎn sʉ́ʉ gaa-fɛɛ gɔ̀ɔn khʉ̂n rót-fai-fáa khâ

S + V2 + หลัง + V1
S + V2 + lǎng + V1
do V2 after V1
 先做 V1，再做 V2；只是句子角度反過來。
ฉันนอนหลังอาบน้ำค่ะ
chǎn nɔɔn lǎng àap-náam khâ

ฉันกินข้าวหลังเลิกเรียนค่ะ
chǎn gin khâao lǎng lə̂ək-rian khâ
I eat after class.
S + V + ก่อน / หลัง + time
S + V + gɔ̀ɔn / lǎng + time
do something before / after a time
 和時間點比較。
ฉันมาถึงโรงเรียนก่อนเก้าโมงค่ะ

理解技巧： ก่อน 看左邊動作先發生； หลัง 看右邊動作先發生。初學時可以先用中文想： 「A ก่อน B」= A 在 B 之前； 「A หลัง B」= A 在 B 之後。

Common Mistakes & Quick Review

Mistake Why it is wrong / risky Better Version
ก่อน / หลัง 方向搞反
gɔ̀ɔn / lǎng
before / after
 A ก่อน B = A 在 B 之前；A หลัง B = A 在 B 之後。
ฉันกินข้าวก่อนเรียนค่ะ
chǎn gin khâao gɔ̀ɔn rian khâ
I eat before studying.
ฉันกินข้าวหลังเรียนค่ะ
chǎn gin khâao lǎng rian khâ
I eat after studying.
สาย = late / line 混淆
sǎai
late / line
 ไปสาย 是遲到；สายสีลม 是 Silom Line。
วันนี้ฉันมาสายค่ะ
wan-níi chǎn maa sǎai khâ
Today I came late.
ฉันขึ้นรถไฟฟ้าสายสีลมค่ะ
chǎn khʉ̂n rót-fai-fáa sǎai sǐi-lom khâ
I take the Silom Line.
ด้วยกัน / พร้อมกัน 混用
dûai-gan / phrɔ́ɔm-gan
together / at the same time
 ด้วยกัน 是一起；พร้อมกัน 是同時。
เราไปกินข้าวด้วยกันค่ะ
rao bpai gin khâao dûai-gan khâ
We go eat together.
เราถึงร้านพร้อมกันค่ะ
rao thʉ̌ng ráan phrɔ́ɔm-gan khâ
We arrived at the restaurant at the same time.
เหมือนกัน / เดียวกัน 混用
mʉ̌an-gan / diao-gan
same/alike / the same one
 เหมือนกัน 是一樣/相似；เดียวกัน 是同一個具體東西。
เราชอบชาไทยเหมือนกันค่ะ
rao chɔ̂ɔp chaa-thai mʉ̌an-gan khâ
We both like Thai tea.
เราอยู่ห้องเดียวกันค่ะ
rao yùu hɔ̂ng diao-gan khâ
We are in the same room.
交通工具量詞混用
khan / lam / khà-buan / sǎai
vehicle / plane-boat / train / line classifier
 車、船、飛機、火車、路線量詞不同。
รถเมล์หนึ่งคัน / เรือหนึ่งลำ / รถไฟหนึ่งขบวน / รถไฟฟ้าสายสีลม
rót-mee nʉ̀eng khan / rʉa nʉ̀eng lam / rót-fai nʉ̀eng khà-buan / rót-fai-fáa sǎai sǐi-lom
one bus / one boat / one train / Silom Line

Street Thai Conversion
