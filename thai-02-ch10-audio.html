Thai Book 2｜Day 10 學習指南

主題：เวลา｜when / while、ก่อน...แล้วก็...ทีหลัง、ฝาก...ไว้ที่ / ไว้กับ、ฝาก買東西、ฝาก帶東西、ของฝาก

Review Common mistake

Lesson Overview

Day 10 的核心有三條線。第一是 เวลา / wee-laa / when / while / time， 用來說「做 A 的時候做 B」。第二是 ก่อน, แล้วก็...ทีหลัง / gɔ̀ɔn, lɛ́ɛw-gɔ̂ɔ...thii-lǎng / first, and then...later， 用來說先後順序。第三是 ฝาก / fàak / to leave something with someone / ask someone to do something / bring something for someone， 這是非常泰國生活化的一個字。

When / while
อย่าเล่นมือถือเวลาขับรถค่ะ
yàa lên mʉʉ-thʉ̌ʉ wee-laa khàp-rót khâ
Don’t use your phone while driving.
First, then later
ลูกทำการบ้านก่อน แล้วก็ดูทีวีทีหลังค่ะ
lûuk tham-gaan-bâan gɔ̀ɔn, lɛ́ɛw-gɔ̂ɔ duu thii-wii thii-lǎng khâ
The child does homework first, and then watches TV later.
Leave something somewhere
ฝากกระเป๋าเดินทางไว้ที่โรงแรมค่ะ
fàak grà-bpǎo-dəən-thaang wái thîi rooŋ-rɛɛm khâ
Leave the luggage at the hotel.
Souvenir
ของฝากจากกรุงเทพฯ
khɔ̌ɔng-fàak jàak grung-thêep
souvenir from Bangkok
Review： Day 9 學過 ก่อน / หลัง / gɔ̀ɔn / lǎng / before / after。 Day 10 的 ก่อน, แล้วก็...ทีหลัง / gɔ̀ɔn, lɛ́ɛw-gɔ̂ɔ...thii-lǎng / first, and then...later 更像「排順序」：先做一件事，之後再做另一件事。
曼谷生活連結： 這課可以直接用在飯店寄放行李、請朋友幫買東西、請人幫帶伴手禮、拜託鄰居看家、提醒朋友開車不要玩手機、飛機起飛時不要滑手機。

Supplement｜本課重點補充

เวลา｜when / while：什麼時候可以用？

Use Thai / Romanization / English Explanation Real-life Example
Two actions happen at the same time
S + V1 + เวลา + S + V2
S + V1 + wee-laa + S + V2
S does V1 when/while S does V2.
 表示「當……的時候」。
ลูกดูทีวีเวลาลูกทำการบ้านค่ะ
lûuk duu thii-wii wee-laa lûuk tham-gaan-bâan khâ
The child watches TV while doing homework.
Things you should not do while doing something
V1 + เวลา + V2 + ไม่ได้
V1 + wee-laa + V2 + mâi-dâai
You cannot do V1 while doing V2.
 教材第 2 頁有一組「不能同時做」的練習。
เล่นมือถือเวลาขับรถไม่ได้ค่ะ
lên mʉʉ-thʉ̌ʉ wee-laa khàp-rót mâi-dâai khâ
You cannot use your phone while driving.
Safety warning
อย่า + V1 + เวลา + V2
yàa + V1 + wee-laa + V2
Don’t do V1 while doing V2.
 比單純說 ไม่ได้ 更像提醒、警告。
อย่าสูบบุหรี่เวลาเติมน้ำมันค่ะ
yàa sùup bù-rìi wee-laa dtəəm náam-man khâ
Don’t smoke while refueling.
容易混淆： เวลา / wee-laa / when / while 不是只表示「時間幾點」。本課多數情況是連接兩個動作，意思接近「當……的時候」。

นี่ของขวัญวันเกิดให้คุณค่ะ
nîi khɔ̌ɔng-khwǎn wan-gə̀ət hâi khun khâ
This is a birthday present for you.
曼谷購物提醒： 買給朋友/家人的泰國小物，像 ยาดม / yaa-dom / Thai inhaler、 ยาหม่อง / yaa-mɔ̀ɔng / balm、 สบู่ / sà-bùu / soap 都很適合說成 ของฝาก / khɔ̌ɔng-fàak / souvenir。

Sentence Structures

Structure｜S + V เวลา S + V：當……的時候

S + V1 + เวลา + S + V2
S + V1 + wee-laa + S + V2
S does V1 when/while S does V2.
Textbook Example Life Variations 中文說明
ลูกดูทีวีเวลาลูกทำการบ้านค่ะ
lûuk duu thii-wii wee-laa lûuk tham-gaan-bâan khâ
The child watches TV while doing homework.
ฉันฟังเพลงเวลาอาบน้ำค่ะ
chǎn fang-phleeng wee-laa àap-náam khâ
I listen to music while showering.
อย่าเล่นมือถือเวลาขับรถค่ะ
yàa lên mʉʉ-thʉ̌ʉ wee-laa khàp-rót khâ
Don’t use your phone while driving.
ฉันชอบคุยโทรศัพท์เวลาดูหนังไม่ได้ค่ะ
chǎn chɔ̂ɔp khui thoo-rá-sàp wee-laa duu nǎng mâi-dâai khâ
I can’t talk on the phone while watching a movie.
ห้ามสูบบุหรี่เวลาเติมน้ำมันค่ะ
hâam sùup bù-rìi wee-laa dtəəm náam-man khâ
Smoking is prohibited while refueling.
 เวลา 接在兩個動作中間，表示「做某事的時候」。

Structure｜V1 ก่อน, แล้วก็ V2 ทีหลัง：先……，再……

S + V1 + ก่อน, แล้วก็ + V2 + ทีหลัง
S + V1 + gɔ̀ɔn, lɛ́ɛw-gɔ̂ɔ + V2 + thii-lǎng
S does V1 first, then does V2 later.
Textbook Example Life Variations 中文說明
ลูกทำการบ้านก่อน แล้วก็ดูทีวีทีหลังค่ะ
lûuk tham-gaan-bâan gɔ̀ɔn, lɛ́ɛw-gɔ̂ɔ duu thii-wii thii-lǎng khâ
The child does homework first, then watches TV later.
เรียนพูดก่อน แล้วก็เรียนเขียนทีหลังค่ะ
rian phûut gɔ̀ɔn, lɛ́ɛw-gɔ̂ɔ rian khǐan thii-lǎng khâ
Study speaking first, then study writing later.
ฝากกระเป๋าก่อน แล้วค่อยไปกินข้าวค่ะ
fàak grà-bpǎo gɔ̀ɔn, lɛ́ɛw khɔ̂i bpai gin khâao khâ
Leave the luggage first, then go eat later.
ซื้อของฝากก่อน แล้วค่อยกลับคอนโดค่ะ
sʉ́ʉ khɔ̌ɔng-fàak gɔ̀ɔn, lɛ́ɛw khɔ̂i glàp khɔɔn-doo khâ
Buy souvenirs first, then go back to the condo.
โอนเงินก่อน แล้วก็ส่งสลิปทีหลังค่ะ
oon ngoen gɔ̀ɔn, lɛ́ɛw-gɔ̂ɔ sòng sà-líp thii-lǎng khâ
Transfer the money first, then send the slip later.
 แล้วค่อย 是更自然口語版，意思是「然後再……」。

Structure｜ฝาก + N + ไว้ที่ / ไว้กับ：寄放在某地 / 某人那裡

ฝาก + N + ไว้ที่ + Place / ฝาก + N + ไว้กับ + Person
fàak + N + wái thîi + Place / fàak + N + wái gàp + Person
leave N at a place / leave N with a person
Textbook Example Life Variations 中文說明
ฝากกระเป๋าเดินทางไว้ที่โรงแรมค่ะ
fàak grà-bpǎo-dəən-thaang wái thîi rooŋ-rɛɛm khâ
Leave the luggage at the hotel.
ฝากของไว้ที่ล็อบบี้ได้ไหมคะ
fàak khɔ̌ɔng wái thîi lɔ́p-bîi dâai mái khá
Can I leave my things at the lobby?
ฝากพัสดุไว้กับนิติได้ไหมคะ
fàak phát-sà-dù wái gàp ní-dtì dâai mái khá
Can I leave the parcel with the juristic office?
ฝากกุญแจไว้กับเพื่อนได้ไหมคะ
fàak gun-jae wái gàp phʉ̂an dâai mái khá
Can I leave the key with my friend?
 ที่ 接地點；กับ 接人或負責方。

Structure｜ฝากซื้อ / ช่วยซื้อ...ให้หน่อย：請人幫買

A ฝาก B ซื้อ + N / ช่วยซื้อ + N + ให้ + A + หน่อยค่ะ
A fàak B sʉ́ʉ + N / chûai sʉ́ʉ + N + hâi + A + nòi khâ
A asks B to buy N / Please buy N for A.
Textbook Example Life Variations 中文說明
ฉันฝากเพื่อนซื้อน้ำหอมค่ะ
chǎn fàak phʉ̂an sʉ́ʉ náam-hɔ̌ɔm khâ
I asked my friend to buy perfume.
ช่วยซื้อยาดมให้ฉันหน่อยค่ะ
chûai sʉ́ʉ yaa-dom hâi chǎn nòi khâ
Please buy Thai inhalers for me.
ฝากซื้อเครื่องสำอางที่สยามให้หน่อยค่ะ
fàak sʉ́ʉ khrʉ̂ang-sǎm-aang thîi sà-yǎam hâi nòi khâ
Please buy cosmetics at Siam for me.
ฝากซื้อขนมไทยกลับมาหน่อยได้ไหมคะ
fàak sʉ́ʉ khà-nǒm thai glàp maa nòi dâai mái khá
Could you buy Thai snacks and bring them back?
 ฝากซื้อ 很像中文「託你買」。

Structure｜ซื้อ N มาฝาก B / ฝาก N ไปให้ C：買來送 / 幫帶給某人

A ซื้อ + N + มา ฝาก + B / A ฝาก + N + ไปให้ + C
A sʉ́ʉ + N + maa fàak + B / A fàak + N + bpai hâi + C
A buys N and brings it for B / A asks someone to take N to C.
Textbook Example Life Variations 中文說明
ฉันซื้อขนมมาฝากเพื่อนค่ะ
chǎn sʉ́ʉ khà-nǒm maa fàak phʉ̂an khâ
I bought snacks for my friend.
ฉันซื้อยาดมมาฝากแม่ค่ะ
chǎn sʉ́ʉ yaa-dom maa fàak mɛ̂ɛ khâ
I bought Thai inhalers for my mom.
ฝากสบู่ไปให้เพื่อนที่ไต้หวันหน่อยค่ะ
fàak sà-bùu bpai hâi phʉ̂an thîi dtâi-wan nòi khâ
Please take the soap to my friend in Taiwan.
ฝากของฝากถุงนี้ไปให้ลูกสาวหน่อยค่ะ
fàak khɔ̌ɔng-fàak thǔng níi bpai hâi lûuk-sǎao nòi khâ
Please take this bag of souvenirs to my daughter.
 มาฝาก 是買來給你；ไปให้ 是帶去給第三人。
